Indie Linux Distros

Indie linux distros are linux distros that are not based off any over linux distro, ie. the only thing that it would have been based off is likely the linux kernel, for the kernel. This includes stuff like Gobo Linux,
Gentoo,
Void Linux.
 These Linux Distros have only maybe grabbed some basic applications and definitely the linux kernel, or it wouldn't even be linux!
