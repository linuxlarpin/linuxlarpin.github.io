Bloatbuntu

I've called it BloatBuntu instead of Ubuntu due to how many pre-installed packages there is an how there is such a heavy DE, heavy file indexing... etc. Another thing is how much an average snap package takes of storage, and the downloads. The average snap package is around 235mb, which would sound small, but on .deb, it would be almost half that.
